# DS-3002 Assignment #4 – Heartbeat to Heatmap

## Project Structure
- `assignment4.py` – run all analysis, generate plots and models.
- `app.py` – Streamlit dashboard.
- `models/` – saved models and artifacts.
- `report/` – final PDF report with screenshots.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run main script: `python assignment4.py`
3. Launch dashboard: `streamlit run app.py`

All random seeds fixed to 42 for reproducibility.
